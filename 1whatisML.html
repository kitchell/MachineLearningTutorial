What is Machine Learning?

Machine learning is a subfield of artificial intelligence that focuses on teaching computers how to learn without being specifically programmed for certain tasks. The idea is to create algorithms that learn from data and make predictions on data. Machine learning can be split into several categories:

  • Supervised learning: the computer is given training data and the desired output from that data (e.g. category labels) and it has to learn from the training data in order to make meaningful predictions based on new data.
  • Unsupervised learning: the computer is given data only and it has to find meaningful structure or groups in the data by itself with no supervision.
  • Semisupervised learning: the computer is given both labeled data and unlabeled data. Labeling large datasets can be time consuming, so typically there is a small amount of labeled data and a large amount of unlabeled data.
  • Reinforcement learning: the computer is interacting with the environment and learning via feedback which behaviors generate rewards.

Supervised Learning

Supervised learning is typcially used for Classification and Regression problems.

Classification

Classification algorithms are used when you have defined labels for your data and you want the model to predict the label of new data. Classification can be binary (only two labels possible for your data) or multi-class (more than 2 possible labels).

Example Algorithms:

  • Logistic Regression
  • Nearest Neighbor
  • Guassian Naive Bayes
  • Decision Trees
  • Support Vector Machine
  • Random Forest

Regression

Regression algorithms are used when you want to predict a continuous value for new data.

Example Algorithms:

  • Linear Regression
  • Decision Trees
  • Nearest Neighbor

Unsupervised Learning

Unsupervised Learning is typically used when you do not have labeled data or you want the model to find patterns, structures, or groups in the data. It is used for clustering and association.

Clustering

Clustering algorithms are used when you want to divide your data into groups where the items in the group have similar traits.

Example Algorithms:

  • K-Means
  • Gaussian Mixture Model
  • DBSCAN (Density-Based Spatial Clustering of Applications with Noise)
  • Agglomerative Hierarchical Clustering

Association

Association rules are used to uncover interesting relationships between data objects in large datasets or databases.

Example Algorithms:

  • Apriori algorithm

Semisupervised Learning

Semisupervised learning is used when there is some labeled data, but not all of it is labeled. The labeled data is used to help identify that there is structure to the data and suggests what it might be and the unlabeled data is used to further refine what that struture is.

Example Methods

  • Pseudo Labeling or Proxy Labeling

Reinforcement Learning

Reinforcement learning algorithms are used when you are able to continually update your model with new data. Typically this will be used with an object that is interacting with its environment (e.g. robot, self-driving car).

Example Algorithms:

  • Temporal Difference
  • Q-learning
